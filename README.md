# Web app Leopardi (Home)

Struttura pronta per GitHub Pages.

## Cosa c’è nella Home
- **Sezione 1:** video introduttivo (file atteso: `assets/intro.mp4`)
- **Sezione 2:** testo lezione caricato da `intro.txt` (scroll)
- **Sezione 3:** area appunti con salvataggio su browser (localStorage) + download in `.txt`
- **Sezione 4:** barra superiore con pulsante **Vedi schema** (apre l’immagine `assets/Home.png`)
- **Sezione 5:** icona **Vita** → `biografia.html`
- **Sezione 6:** icona **Romanticismo** → `romanticismo/romanticismo.html`
- **Sezione 7:** icona **Pessimismo storico** → `storico/storico.html`
- **Sezione 8:** icona **Pessimismo cosmico** → `cosmico/cosmico.html`

## Come personalizzare
1. **Sostituisci il testo**: modifica `intro.txt`.
2. **Aggiungi il video**: copia un mp4 in `assets/intro.mp4`.
3. **Riempi le pagine**: modifica `biografia.html`, `romanticismo/romanticismo.html`, `storico/storico.html`, `cosmico/cosmico.html`.

## Nota sugli appunti
Gli appunti vengono salvati nel browser del dispositivo: se uno studente cambia browser o cancella i dati del sito, gli appunti non saranno più disponibili.
